Basic CSS Properties

These are the most commonly used properties when starting out with CSS.

  • color: Defines the text color.
  • background-color: Sets the background color of an element.
  • font-size: Adjusts the size of the font.
  • font-family: Specifies the font to be used.
  • margin: Defines the space around an element.
  • padding: Sets the space between the content and its border.
  • border: Adds a border around an element (e.g., border: 1px solid black).
  • width / height: Defines the width and height of an element.
  • text-align: Aligns text inside an element (left, right, center, justify).
  • display: Defines how an element is displayed (block, inline, flex, none, etc.).
  • background-image: Sets a background image for an element.
  • line-height: Controls the space between lines of text.
  • list-style: Specifies the type of list item marker (circle, square, etc.).
  • opacity: Defines the transparency of an element.
  • overflow: Controls the overflow behavior (hidden, scroll, visible, auto).

Medium CSS Properties

These are properties used for more complex layouts and design improvements.

  • position: Specifies the positioning of an element (relative, absolute, fixed, sticky).
  • z-index: Controls the stacking order of positioned elements.
  • float: Positions elements side by side or moves them left or right.
  • flex: Used to create flexible layouts with Flexbox.
  • grid: Defines a grid-based layout system.
  • align-items / justify-content: Used in Flexbox/Grid for alignment.
  • transform: Applies 2D or 3D transformations to elements (e.g., rotate, scale).
  • transition: Adds smooth transitions between CSS property changes.
  • box-shadow: Adds shadow effects around elements.
  • text-shadow: Applies shadows to text.
  • media queries: Used for responsive design by applying different styles based on screen size or device type.
  • object-fit: Controls how content (like images) fits inside its container (fill, contain, cover).
  • vertical-align: Aligns inline or table-cell elements vertically.
  • cursor: Specifies the type of cursor to display when hovering over an element.
  • background-size: Adjusts the size of background images (cover, contain).

Advanced CSS Properties

These properties are often used in more complex and responsive design solutions.

  • clamp(): A function that allows setting a CSS property to scale between a minimum, maximum, and preferred value.
  • min() / max(): Functions that return the minimum or maximum of two or more values (e.g., width: min(100%, 600px)).
  • aspect-ratio: Sets the aspect ratio of an element, helping maintain proportions.
  • calc(): Allows performing calculations to determine CSS values dynamically (e.g., width: calc(100% - 50px)).
  • @supports: Used for feature queries to apply styles if a browser supports certain CSS properties.
  • filter: Applies graphical effects like blur, brightness, contrast, etc., to elements (e.g., filter: blur(5px)).
  • backdrop-filter: Applies graphical effects to the area behind an element.
  • grid-template-areas: Used for creating layouts with named grid areas.
  • content-visibility: Optimizes performance by skipping rendering work for off-screen content until it is needed.
  • scroll-snap: Defines scroll snapping behavior to align content when scrolling.
  • object-position: Aligns replaced elements (like images or videos) inside their box.
  • isolation: Defines whether an element creates a new stacking context.

  • will-change: Hints to the browser what property of an element will change, allowing for optimization.
  • contain: Limits the scope of an element's layout, style, and paint containment to optimize rendering performance.
